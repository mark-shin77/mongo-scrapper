# Mongo Scraper

## Demo
Indeed Web Dev Scraper is deployed on Heroku! Click <a href="https://indeed-webdev-scraper.herokuapp.com/">here</a> to try it out!

## App Description
An application which scrapes local web developer jobs from indeed into a local database. After the information is scraped 
into the local database, users can save jobs which interest them and add notes to keep track of what interested them about the specific job post.

## Technologies Used
- Express
- Node 
- Mongo DB
- Javascript 
- CSS 
- Handlebars
- Bootstrap
